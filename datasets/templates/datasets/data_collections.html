Dataset collections {% fontawesome_icon 'plus-square' color='#336699' %} new

A Dataset Collection is a set of public datasets—typically from multiple contributors, and linked to define a "focus domain" within World Historical Gazetteer. As with all collections, searches, downloads and API calls can be constrained to include only places in the collection's member datasets.

Place collections {% fontawesome_icon 'plus-square' color='#336699' %} new

A Place Collection is an authored data publication. It can include all places from one or more datasets, as well as individual places from any public dataset discovered in WHG searches. Its place records can be annotated with information pertaining to its membership in the collection. more... {% comment %}

Metadata for collection itself will include an abstract, and can include any number of links to relevant web resources.

{% endcomment %}
Region collections {% fontawesome_icon 'plus-square' color='#336699' %} new

Region collections are composed of all public WHG place records that lie within any area defined in the WHG system. This includes "stock" regions, e.g. from the United Nations, or modern countries, public region features contributed by users, or one of a user's own study areas created on the fly.
