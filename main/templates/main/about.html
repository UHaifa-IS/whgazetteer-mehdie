The MEHDIE Data Integration System

Project Vision

This site is our work-in-progress implementation of the MEHDIE vision for data-driven research by humanities scholars. The site is based upon the World Historical Gazetteer platform. Here, we focus on the middle east and its language and sources, adding two substantial tools to the basic tools provided by the WHG website:.

  • An entity resolution system that attempts to create relations between entities (people, places, events) from different data sources.
  • A data-driven query system that allows researchers to answer complex questions on the combined data.
