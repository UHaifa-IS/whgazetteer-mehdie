Updating datasets

Not yet available

Datasets can be updated by uploading a new file, the contents of which will replace the previous. Each new record is considered by the upload script in turn:

  • If it has a feature/row identifier ('src_id') matching one in the WHG database, it will replace the existing record. However, existing associated place_link and place_geom records that were added during previous reconciliation steps will remain unchanged.
  • If it has a feature/row identifier ('src_id') that does not exist, it will be added to the database. Such records will need to be reconciled against Wikidata, then accessioned as before -- that is, reconciled against the WHG index).
